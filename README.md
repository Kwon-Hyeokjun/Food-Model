# 오픈소스 개발과 활용 기말 프로젝트
한양대 에리카 식당 목록 정리
<br>

---
## 프로젝트 소개
한양대 ERICA 앞 한양대학로에 분포한 다양한 인근 식당과 밥집, 술집들을 소개합니다.
학교 재학생, 대학원생, 교직원 등 학교 관계자 뿐만 아니라 외부인들도 즐거운 캠퍼스 라이프를 즐기도록 도와주기 위해 간편하고 직관적인 UI를 제공하고자 식당 정보를 작성합니다.
한식, 양식, 일식, 중식 등 식당의 종류를 알려주고, 가격대로 분류하여 원하는 식당을 빠르게 찾고, 무엇을 먹을지 고민이 될 때에 도움을 주고자 계획했습니다.
<br>

---

## 주요 기능 및 목표
종류와 가격대 별로 분류하여 누구나 빠르게 원하는 식당을 찾을 수 있도록 하는 것을 목표로 합니다.

**종류** : 분식, 간편식, 야식, 동남아, 일식, 중식, 양식, 한식, 술집로 대분류를 나눈 후, 소분류로 세분화하고 분류에 따른 세부 메뉴를 소개합니다. 

**가격대** : 1만원 이하, 2만원 이하, 3만원 이하, 4만원 이하, gram당 가격(₩)으로 분류됩니다.

## 주요 기능 (Features)
-  **식당 정보 분류 제공**  
  - 대분류(한식/일식/중식/양식/간편식/야식/술집 등)
  - 세부 메뉴(떡볶이, 라멘, 마라탕, 치킨 등)
  - 가격대(1만원 이하, 2만원 이하, g당 가격 등)

-  **필터 기반 탐색 기능**  
  - 가격대별 필터
  - 음식 종류별 필터
  - 가게명 검색 기능 (추가 예정)
