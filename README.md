# 오픈소스 개발과 활용 기말 프로젝트
한양대 에리카 식당 목록 정리 : 'HY-ERICA 식당 검색기'
<br>

---
## 프로젝트 소개
한양대 ERICA 앞 한양대학로에 분포한 다양한 인근 식당과 밥집, 술집들을 검색하는 엔진을 제공합니다. 학교 재학생, 대학원생, 교직원 등 학교 관계자 뿐만 아니라 외부인들도 즐거운 캠퍼스 라이프를 즐기도록 도와주기 위해 간편하고 직관적인 UI를 제공하고자 식당 정보를 작성했습니다. 한식, 양식, 일식, 중식 등 식당의 종류를 알려주고, 가격대로 분류하여 원하는 식당을 빠르게 찾고, 무엇을 먹을지 고민이 될 때에 도움을 주고자 계획했습니다. 또한 평소에 자주 가는 식당 정보에 빠르게 접근할 수 있는 기능을 담았습니다. 이로써 사용자가 원하는 가격, 카테고리, 메뉴 취향에 따라 손쉽게 식당을 선택할 수 있습니다.
<br>

---
## 주요 기능 및 목표
종류와 가격대 별로 분류하여 누구나 빠르게 원하는 식당을 찾을 수 있도록 하는 것을 목표로 합니다.

**종류** : 분식, 간편식, 야식, 동남아, 일식, 중식, 양식, 한식, 술집로 대분류를 나눈 후, 소분류로 세분화하고 분류에 따른 세부 메뉴를 소개합니다. 

**가격대** : 1만원 이하, 2만원 이하, 3만원 이하, 4만원 이하, gram당 가격(₩)으로 분류됩니다. 

**즐겨찾기** : 즐겨찾기('찜') 기능을 이용하여 평소에 자주 가는 식당 목록을 별도로 확인할 수 있습니다.

**키워드** : 키워드로 식당을 검색합니다.

**랜덤추천** : 현재 선택된 필터 조건 내에서 무작위 매장 한 곳을 추천합니다.

**지도연동** : 식당을 클릭하면 자동으로 식당의 위치 정보를 제공합니다.

## GitHub 전체 개발 과정
1. 개발 초기 : 프로젝트 준비 단계

    GitHub 계정 생성 및 SSH/HTTPS 연결 이해

    로컬 폴더 git init 실습

    origin 추가, main 브랜치 정비

    첫 commit/push 성공

2. 개발 중 : 기능 구현 & 버전 관리

    feature 단위로 브랜치 생성하여 기능 개발

    코드 수정 후 git add → git commit → git push

    다음 문제들 발생 : repository not found, no changes added to commit, 로컬 경로와 원격 경로 충돌, 브랜치 이름 mismatch, 상위 폴더가 서브모듈 취급되는     문제

    → Git 구조를 이해하면서 문제 해결

3. GitHub를 통한 협업

    팀원이 데이터, UI, 문서, 기능을 각자 push

    pull/push 시 충돌 해결

    README 업데이트와 코드 개선 과정 기록

    GitHub Actions / Pages 배포 가능 구조 유지

4. 최종 단계 : 문서화 & 배포 준비

    README 문서 최종 정리

    프로젝트 사용 방법, 기술 스택, 기여자 내역 기록
